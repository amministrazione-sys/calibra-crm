☁ Sincronizzazione Google Drive

Come funziona: note, dilazioni e storico solleciti vengono salvati in calibra_crm_data.json sul tuo Google Drive. Da qualsiasi PC apri questo file HTML, ti connetti, e i dati sono sincronizzati.
Configurazione iniziale (una tantum):
1. Vai su console.cloud.google.com con amministrazione@calibrapa.com
2. Crea progetto → es. calibra-crm
3. API e servizi → Libreria → abilita Google Drive API e Gmail API
4. Credenziali → Crea → OAuth 2.0 Client ID → Applicazione web
5. In Origini JavaScript autorizzate aggiungi: null
6. Copia il Client ID e incollalo qui sotto
Il Client ID viene salvato solo localmente, non viene mai trasmesso ad altri server.

🔑 Incolla il token di autorizzazione

Si è aperta una nuova scheda con la schermata di consenso Google.

1. Accedi con amministrazione@calibrapa.com e autorizza
2. Il browser ti porta su una pagina localhost con errore — è normale
3. Copia l'intero URL dalla barra del browser di quella scheda
4. Incollalo qui sotto e clicca Connetti
